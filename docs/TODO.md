# Z-FFA TODO / Roadmap

This file tracks ideas that are planned but not yet implemented.

## Next UI Features

- Armour trims shop
  - A cosmetic/tier showcase menu where trims symbolize rank or progression.
  - Should feel separate from the main queue menus.
  - Could be used to show progression, status, or unlock tiers.

- Custom kit editor
  - A separate menu where players can build their own kits.
  - Players should be able to save, preview, and equip custom kits.
  - Future options could include:
    - private use only
    - public showcase
    - challenge others using the custom kit
    - allow other players to use or copy the kit

## Possible Follow-Ups

- Custom kit sharing / copying system
- Kit showcase menu
- Cosmetic tier rewards linked to ranks
- Trim-based status display in lobby menus
- Rating or voting system for player-made kits

## Notes

- Keep this separate from queue matchmaking UI.
- Custom kit editing should probably live in its own menu flow.
- If implemented later, it should store data separately from default kits.
